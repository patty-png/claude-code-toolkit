Top 10 Most Impactful — With Install Guides

If you install only ten things, make them these. Ordered by real-world leverage, not novelty.

#1

GitHub MCP Server

Official

51 tools for repos, issues, PRs, Actions, releases. The single highest-leverage MCP — #1 most-used across every 2026 dev survey (28k+ stars). Pairs natively with Claude Code's file access.

Install (remote, OAuth — recommended)

claude mcp add --transport http github https://api.githubcopilot.com/mcp/

Install (local, token)

claude mcp add github -- docker run -i --rm \
  -e GITHUB_PERSONAL_ACCESS_TOKEN=ghp_xxx \
  ghcr.io/github/github-mcp-server
#2

Filesystem MCP

Official · Free

Gives Claude sandboxed read/write across any folders you allow — beyond the current working directory. The cheapest, most immediate productivity bump.

Install

claude mcp add filesystem -- npx -y @modelcontextprotocol/server-filesystem \
  "C:/Users/johnd/Personal" \
  "C:/Users/johnd/Downloads"
#3

Playwright MCP

Microsoft

Accessibility-tree browser automation — no screenshot guessing. Lets Claude test web apps, scrape authed pages, fill forms. Gold standard for browser work.

Install

claude mcp add playwright -- npx -y @playwright/mcp@latest
npx playwright install chromium
#4

Context7 MCP

Upstash

Fixes the #1 Claude Code failure mode: outdated library knowledge. Injects live, version-pinned docs for 10k+ libraries on demand. Add use context7 to any prompt.

Install (remote)

claude mcp add --transport http context7 https://mcp.context7.com/mcp
#5

Sentry MCP

Remote · OAuth

Error-tracking directly in the loop. Ask Claude to pull the top 5 unresolved errors, triage root cause via Seer AI, then patch. Zero-install remote server.

Install

claude mcp add --transport http sentry https://mcp.sentry.dev/mcp
#6

Superpowers (obra)

Skill Framework

The largest community skill library. A composable SDLC framework: brainstorm → worktree → plan → TDD execute → review → merge, all as chained skills.

Install

cd ~/.claude/plugins
git clone https://github.com/obra/superpowers.git
# then /plugin enable superpowers inside Claude Code
#7

wshobson/agents

100+ Subagents

Production-ready subagents for orchestration: backend-architect, database-architect, security-auditor, test-automator, deployment-engineer, etc. Drop in as a plugin.

Install

mkdir -p ~/.claude/agents
git clone https://github.com/wshobson/agents ~/.claude/agents/wshobson
# subagents auto-discover on next Claude Code start
#8

Notion MCP

Official Remote

18 tools for pages, databases, search. Turns Notion into Claude's long-term memory — dump meeting notes, research, specs and let Claude query them mid-task.

Install

claude mcp add --transport http notion https://mcp.notion.com/mcp
# OAuth flow opens automatically on first use
#9

n8n-mcp (czlonkowski)

Automation Bridge

Lets Claude Code build n8n workflows from plain English (1,500+ node knowledge). The cleanest answer to "replace my Zapier stack." 90–100% workflow completion in one session.

Install

claude mcp add n8n -- npx -y n8n-mcp
# docs: https://github.com/czlonkowski/n8n-mcp
#10

NotebookLM + Claude Code Combo

Workflow

Not a tool — a pattern. NotebookLM for grounded research across PDFs, videos, docs. Export key findings → paste into Claude Code as .claude/context/ files. Blind-spot coverage.

Setup

# 1. notebooklm.google.com — create notebook, add sources
# 2. Use "Briefing doc" or "Study guide" to distill
# 3. Copy output to ./docs/research-brief.md in your repo
# 4. Claude Code auto-reads via CLAUDE.md context include
Install locations. MCPs added with claude mcp add default to user scope — available in every project. Add --scope project to pin an MCP to a single repo (writes to .mcp.json). Add --scope local for machine-only config not committed to git.

Recommended Stack Strategy for You

Given your profile — Python primary, Windows + Mac, finance/health/client projects, existing Coinbase/Plaid/Wealthsimple integrations:

Phase 1 — This Week (high-leverage, zero-friction)

Install in order: GitHub MCP → Filesystem MCP → Context7 → Sentry MCP → Notion MCP. All are remote or npx — under 15 minutes total.

Phase 2 — Next (domain-specific power-ups)

For your finance panel: add Plaid MCP + Coinbase MCP so Claude can debug live data. For Jarvis/projects: add Playwright MCP + n8n-mcp. Install wshobson/agents to get parallel specialists.

Phase 3 — Workflow Polish

Adopt Superpowers skill framework for larger features. Add a Prettier/Black PostToolUse hook. Wire Claude Code Stop event to a desktop notification on Windows via PowerShell. Start using NotebookLM for any domain research before coding.

Avoid over-installing. Every MCP loaded = tokens eaten on startup + tool-choice confusion. Stick to the Top 10, add domain-specific servers only when actively working on that domain. Use claude mcp list and claude mcp remove <name> regularly.
